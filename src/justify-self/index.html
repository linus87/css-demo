Features

  • The justify-self property aligns a grid item within its grid cell in the inline direction.
  • It applies to block-level boxes, absolutely-positioned boxes, and grid items
  • If parent is display:flex, display:block, justify-self won't work.
  • justify-items defines the default justify-self for all items in the box, it's used on parent element
